# Velox Engine


## Build from source

#### Prerequisites
- LLVM (Clang & Clang++)
- Ninja
- CMake
- Vulkan SDK

```powershell
git clone https://github.com/Georgefwm/VeloxEngine
```
```powershell
cd VeloxEngine
```
(only tested on windows)
To build, simply:
```powershell
./build
```
```powershell
./run
```
To run tests:
```powershell
./build_tests
```
```powershell
./test
```
