# Velox Engine


## Build from source

#### Prerequisites
- Clang compiler (LLVM)
- Ninja
- CMake

```shell
git clone https://github.com/Georgefwm/VeloxEngine
```
```shell
cd VeloxEngine
```
(only tested on windows)
To build, simply:
```shell
./build.cmd
```
To run:
```shell
./run.cmd
```
